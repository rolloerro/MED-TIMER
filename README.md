🩺 MED-TIMER — Telegram бот напоминаний о приёме лекарств

MED-TIMER — простой и удобный Telegram-бот, который помогает не забывать о приёме лекарств.
Создан как лёгкий MVP: быстро, понятно, без перегруза.

🚀 Возможности
⏰ Напоминания о приёме лекарств
📋 Список всех активных напоминаний
❌ Очистка напоминаний 
⚡ Мгновенный запуск без регистрации

🧠 Минимальный интерфейс — максимум пользы

🧠 Как это работает
Пользователь нажимает «Добавить приём»
Вводит название лекарства и время (в минутах)
Бот присылает напоминание точно в срок
Всё работает локально, без базы данных (MVP-логика)

🛠️ Технологии
Python 3.12
python-telegram-bot v20+
Async / asyncio
ENV-переменные для токена
Минимум зависимостей

📦 Установка и запуск
1️⃣ Клонируем репозиторий
git clone https://github.com/rolloerro/MED-TIMER.git
cd MED-TIMER

2️⃣ Создаём виртуальное окружение
python3 -m venv venv
source venv/bin/activate

3️⃣ Устанавливаем зависимости
pip install -r requirements.txt

4️⃣ Создаём .env
BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER

5️⃣ Запуск
python bot.py
